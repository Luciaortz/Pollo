#  Pollo - Página Web Informativa

Aplicación web desarrollada con HTML, CSS y JavaScript que presenta contenido de forma sencilla e interactiva mediante una interfaz web ligera y fácil de mantener.

---

##  Descripción

Pollo es un proyecto web desarrollado como práctica de desarrollo frontend utilizando tecnologías básicas de la web. La aplicación permite mostrar contenido a través de una interfaz sencilla, aplicando conceptos fundamentales de estructura HTML, estilos CSS y programación con JavaScript.

El proyecto fue diseñado para fortalecer conocimientos relacionados con la construcción de páginas web estáticas, organización de archivos y manipulación básica del comportamiento de una aplicación web desde el navegador.

---

##  Características

* Interfaz web ligera y sencilla.
* Diseño basado en HTML y CSS.
* Funcionalidades implementadas con JavaScript.
* Estructura organizada para facilitar el mantenimiento.
* Ejecución directa desde el navegador.
* Proyecto ideal para prácticas de desarrollo frontend.

---

##  Tecnologías Utilizadas

* HTML5
* CSS3
* JavaScript

Permite visualizar el contenido principal de la aplicación mediante una página web desarrollada con HTML.

### Comportamiento Dinámico

##  Instalación

### 1. Clonar el repositorio

```bash
git clone https://github.com/usuario/Pollo.git
```

### 2. Acceder al directorio del proyecto

```bash
cd Pollo-master
```

### 3. Abrir el archivo principal

```text
main.html
```

O ejecutar el proyecto utilizando Live Server desde Visual Studio Code.
